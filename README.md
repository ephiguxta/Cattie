# Cattie

![Cattie](img/cattie.png)

cattie is a logic game for Windows and Linux.

# Compilation - Steps

## For Windows

```bash
.\scripts\mkall_win32.bat
```

## For Linux
```bash
$ bash ./scripts/mkall_linux.sh
```

# Instalation

## For Windows

```bash
.\scripts\install_win32.bat
```

## For Linux

```bash
$ sudo ./scripts/install_linux.sh
```
